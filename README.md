#  Flipkart Product Data Analysis

This project performs Exploratory Data Analysis (EDA) on a Flipkart dataset using Python.

##  Features
- Data Cleaning using Pandas
- NumPy operations
- Visualization using Matplotlib & Seaborn
- Insights from product data

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Google Colab

##  Key Insights
- Top product categories identified based on ratings
- Price vs rating analysis
- Category distribution visualization
